# Session Wrap-Up

Run this at the end of every session to capture what happened and commit it to long-term memory.

## Step 0: Ensure AI Brain Notebook Exists

Check if the user already has a Brain notebook set up by looking for a memory file that stores the Brain notebook ID.

## Step 1: Review the Session

Look back and identify: decisions made, work completed, key learnings, open threads, user preferences.

## Step 2: Save Memories

Update memories for: feedback, project, user, reference.

## Step 3: Write Session Summary

Create markdown with: What We Did, Decisions Made, Key Learnings, Open Threads, Tools & Systems Touched.

## Step 4: Push to NotebookLM Brain

Add session summary as source to Brain notebook.

## Step 5: Confirm

Tell the user the status.

## Prerequisites

Requires NotebookLM CLI installed and authenticated.
